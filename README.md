#### Genesis Portfolio Template

### How to run

```
git clone https://github.com/korayguler/genesis-portfolio-template
```

```
cd genesis-portfolio-template
```

```
npm install
```

```
npm start
```

#### Tech Stack

```
- html
- scss
- javascript
- gulp.js
- font-awesome
- image-hover
```
